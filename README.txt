Dies ist eine CLI Programm für Risiko. Es unterstützt dasselbe Brettspiel auf folgende Weise:
  1. Speichern aller Spieldaten
  2. Führung des Spiels
  3. Würfeln
  4. Ermitteln des Gewinners
 
Um das CLI Programm zu starten, öffne die Eingabeaufforderung, navigiere in den Ordner Risiko und führe main.py aus
Wenn du anschließend den Befehl Spiel Starten eingibst wird ein neues Spiel gestartet
